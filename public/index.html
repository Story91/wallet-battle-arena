🤖 MCP Tool: wallet_battle

This battle arena runs as an MCP server (JSON-RPC 2.0 over stdio). AI agents can call wallet_battle to compare wallets programmatically. Payments work via the x402 protocol — just include the X-Payment header.

// MCP tool call
{ "method": "tools/call",
  "params": { "name": "wallet_battle",
    "arguments": {
      "wallet_a": "0xd8dA...",
      "wallet_b": "0x1234...",
      "chain": "base"
    }
  }
}
